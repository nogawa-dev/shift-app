# 塾配布シフト管理システム

塾スタッフのシフト希望を収集・管理するWebアプリです。

## 機能

- **シフト提出**：スタッフが希望日・時間帯を選択して一括提出
- **カレンダー表示**：FullCalendarで提出済みシフトを月表示
- **シフト確定**：管理者・オーナーがカレンダー上でクリックして確定
- **募集期間設定**：管理者が開始日〜終了日を指定（土日・祝日を自動除外）
- **ユーザー管理**：オーナーがスタッフのID・権限・名前を管理
- **セッションタイムアウト**：30分操作なしで自動ログアウト

## 権限

| 権限 | できること |
|------|-----------|
| 👑 オーナー | 全操作 + ユーザー登録・削除・ID変更 |
| 🛡️ 管理者 | シフト確定 + 募集期間設定 |
| 👤 一般ユーザー | シフト提出・取り消し |

## 時間帯

| 番号 | 時間 |
|------|------|
| 1 | 7:30 〜 8:30 |
| 2 | 8:00 〜 9:00 |
| 3 | 15:30 〜 16:30 |
| 4 | 17:30 〜 18:30 |

## 技術スタック

- **バックエンド**：Python / Flask
- **データベース**：Supabase（PostgreSQL）
- **フロントエンド**：HTML / CSS / JavaScript
- **ライブラリ**：FullCalendar v6、SweetAlert2、jpholiday

## セットアップ

```bash
# 依存パッケージのインストール
pip install -r requirements.txt

# ローカル起動（ブラウザが自動で開きます）
python app.py
```

## 環境変数（本番環境）

| 変数 | 説明 |
|------|------|
| `VERCEL` | 設定されている場合、ブラウザ自動起動をスキップ |

## ファイル構成

```
my_app/
├── app.py                  # Flaskメインアプリ
├── requirements.txt        # 依存パッケージ
└── templates/
    ├── base.html           # 共通レイアウト（ヘッダー・ナビ）
    ├── index.html          # シフト入力画面
    ├── calendar.html       # カレンダー表示
    ├── register.html       # ユーザー管理（オーナー専用）
    ├── settings.html       # 募集期間設定
    ├── shifts.html         # シフト一覧
    ├── login.html          # ログイン
    ├── setup.html          # 初回セットアップ
    └── forgot.html         # パスワード再設定
```
